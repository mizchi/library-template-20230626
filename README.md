# @mizchi/library-template

minimum library template (for myself)

## Stack

- pnpm
- typescript
- rome (format on vscode)
- vitest (in-source testing)
- GitHub Actions CI

## How to use

```bash
$ npx degit mizchi/library-template-20230626 <dir>
$ cd <dir>
$ pnpm install
$ pnpm test
$ pnpm build
```
